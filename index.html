Resumen

En el presente Trabajo Terminal se plantea como propuesta, la realizacion de una API que permita desarrollar sistemas de recomendación, la cual brindará funciones de abstracción de datos, clasificación y análisis de los mismos de acuerdo a la evolución de la información recopilada, las características de diferentes objetos y relaciones importantes entre objetos similares.


Sign up today

Propuesta de solución

La Creación de una API que simplifique la creación de sistemas de recomendación para diversos tipos de artículos, con un modelo propuesto por el equipo de trabajo, y la implementación de algoritmos de recomendación para diversos propósitos que engloban el comportamiento de un sistema de recomendación híbrido.



Sign up today

Objetivo

Diseñar e implementar una API que brinde funciones de abstracción y operación de datos, clasificación y análisis de los mismos a través de las características de un conjunto de datos para obtener, por medio de evaluaciones y predicciones, conjuntos de objetos que representen una recomendación.







Algunas tecnologías usadas son:


Spring - Hibernate

Java - Neo4j

Maven - Backbone

JS- JQuery
